Welcome to Ollama Code Review, !

Thank you for installing the Ollama Code Review extension. This extension brings AI-powered code reviews to your Azure DevOps pull requests using self-hosted Ollama models.

Getting Started

Configure the extension in your pipeline YAML to start reviewing code automatically:

  • Set up your Ollama endpoint
  • Choose your preferred AI model
  • Configure review options (bugs, performance, best practices)
  • Optional: Add Bearer token for secured endpoints

For detailed instructions, visit our GitHub repository.
